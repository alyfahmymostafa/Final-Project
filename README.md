Endless Runner – Final Project

Overview This project is a 3D endless runner built in Unity. The player automatically runs forward, switches lanes, jumps over obstacles, and collects coins to increase their score. The game includes a full UI system, audio, animations, lighting, and ScriptableObject-driven data for collectibles.

Features

Forward auto-run movement

Lane switching and jumping

Random obstacle spawning

Collectible coins using ScriptableObject data

Score and time tracking

Game Over screen with restart button

Pause menu with a working volume slider

Main menu with a description panel

Background music and sound effects

Baked lighting and a complete 3D environment

Controls 
Move Left: Left Arrow 
Move Right:Right Arrow 
Jump: Space 
Pause: Escape

How to Play Start the game from the main menu. Avoid obstacles by switching lanes or jumping. Collect coins to increase your score. Survive as long as possible. When you die, the Game Over screen will appear and you can restart.

Build Instructions Download the repository. Open the Builds/Windows folder. Run FinalProject.exe.. No installation is required.

Unity Version This project was created using Unity 2022/2023/6000.x (use your actual version).
